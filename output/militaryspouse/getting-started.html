This page is for after MilitarySpouseTasksAI is installed. Use it to confirm the connection, run your first workflow, understand credits, and fix the most common first-use issues.

First, confirm it is connected

Open the AI assistant where you installed MilitarySpouseTasksAI and start a new conversation. Then ask:

Check my MilitarySpouseTasksAI credit balance.
If your balance appears, the connection is working. Searches, category browsing, and balance checks do not use a credit.
If the assistant says it cannot find MilitarySpouseTasksAI, fully quit and reopen the AI assistant. Many MCP clients load tools only when the app starts.

Try a first workflow

You do not need to know the exact workflow name. Describe the task in ordinary language and ask MilitarySpouseTasksAI to search first.

Use MilitarySpouseTasksAI to search for a workflow about Write a professional license transfer request.
Use MilitarySpouseTasksAI to show me available categories.
Use MilitarySpouseTasksAI to help with Write a professional license transfer request.

MilitarySpouseTasksAI will return matching workflows. Pick the one you want to run. A credit is used when a licensed workflow is launched, not when you search or browse.

Credits and account approval

One credit runs one licensed workflow. Balance checks, category browsing, and search are free.

When installation or account approval is needed, use the email address for your MilitarySpouseTasksAI account. Manual license-key entry is only a backup path.

Privacy in plain English

MilitarySpouseTasksAI servers handle account approval, credits, catalog/search metadata, and licensed workflow delivery. MilitarySpouseTasksAI does not receive your task prompts, uploaded documents, or task answers.

After the licensed workflow is delivered, your chosen AI assistant runs the workflow according to that assistant provider's privacy terms and settings.

If something is not working

The assistant says npm is unavailable

Install Node.js first. Node.js is a standard helper app used by many modern tools, and it includes npm automatically. Go to nodejs.org, download the recommended installer for your computer, run it, reopen your AI assistant, and paste the install prompt again.

MilitarySpouseTasksAI does not appear in your AI assistant

Fully quit and reopen the assistant. If it still does not appear, go back to the install page and paste the install prompt again so the assistant can re-check the setup.

The assistant asks for approval

Approve MilitarySpouseTasksAI when your AI assistant asks. In Claude Desktop, you may also need to set the MilitarySpouseTasksAI tools to always allow in the connector settings.
