The realtortasksai/mcp/ folder is a Python MCP server compatible with Claude Desktop, Cursor, Windsurf, and any MCP stdio client.

Requirements: Python 3.10+  (download)

Option A — Automatic installer (recommended)

Detects and configures all installed MCP clients automatically:

cd ~/Downloads/realtortasksai/mcp pip3 install -r requirements.txt python3 install.py

Restart your MCP client — look for the realtortasksai tools.

Option B — Manual (Claude Desktop)

cd ~/Downloads/realtortasksai/mcp pip3 install -r requirements.txt

Open your config file:

Mac: ~/Library/Application Support/Claude/claude_desktop_config.json Windows: %APPDATA%\Claude\claude_desktop_config.json

Add to "mcpServers":

{ "mcpServers": { "realtortasksai": { "command": "python3", "args": ["/full/path/to/realtortasksai/mcp/server.py"], "env": { "REALTORTASKSAI_LICENSE_KEY": "YOUR_LICENSE_KEY" } } } }

Replace the path and key with your actual values, then restart your client.

Option B — Manual (Cursor / Windsurf)

Go to Settings → MCP Servers and add the same JSON block above.
