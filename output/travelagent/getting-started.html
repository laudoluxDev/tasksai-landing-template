Requirements

  • Python 3.8 or later — download at python.org if needed

2

Download Your Package

After purchase, you'll receive an email with a personalized download link. Your license key is pre-configured in the package — no copying and pasting required.

Extract the zip file. You'll see two folders:

  • openclaw/ — for OpenClaw users
  • mcp/ — for Claude Desktop, Cursor, and Windsurf
3

Run the Installer

Open a terminal (Mac: Terminal app  |  Windows: Command Prompt or PowerShell) and run:

# Navigate to the mcp folder
cd ~/Downloads/travelagent/mcp

# Run the installer
python3 install.py

The installer will:

  1. Detect your installed MCP clients (Claude Desktop, Cursor, Windsurf)
  2. Install the required Python packages automatically
  3. Back up your existing MCP config (nothing is lost)
  4. Configure TravelAgentTasksAI in all detected clients
4

Restart Your MCP Client

Fully quit and reopen Claude Desktop, Cursor, or Windsurf. MCP servers are loaded at startup — a restart is required.

You should see travelagent appear in your connected tools list.

Manual Configuration

If you prefer to configure manually, add this to your MCP client's config file:

{
  "mcpServers": {
    "travelagent": {
      "command": "python3",
      "args": ["/path/to/mcp/server.py"],
      "env": {
        "TRAVELAGENTTASKSAI_LICENSE_KEY": "your_license_key_here"
      }
    }
  }
}

Config file locations:

  • Claude Desktop (Mac): ~/Library/Application Support/Claude/claude_desktop_config.json
  • Claude Desktop (Windows): %APPDATA%\Claude\claude_desktop_config.json
  • Cursor / Windsurf: Check Settings → MCP
