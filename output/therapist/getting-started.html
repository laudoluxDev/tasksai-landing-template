Download Claude Desktop free (Mac & Windows) →
1

Close Claude Desktop

If Claude Desktop is running, fully quit it first — not just close the window.

  • Windows: Right-click the Claude icon in the system tray (bottom-right near the clock) → Quit
  • Mac: Click Claude → Quit Claude in the menu bar, or press ⌘Q

This ensures the installer can write files without conflicts.

2

Download & run the installer

Use the green download button above. Double-click TherapistTasksAI-Setup.exe (Windows) or TherapistTasksAI-Setup (Mac).

  1. Enter your license key (from your signup email)
  2. Click Verify to confirm your key works
  3. Click Install
  4. Wait for "Installation complete" — then close the installer

🛡️ Windows users: you'll see a safety warning — here's what to do

Windows shows a blue "Windows protected your PC" screen for all new software. It doesn't mean anything is wrong — just click through it:

  1. Click "More info" (small text link below the main message)
  2. Click "Run anyway"
  3. The installer opens normally — continue from there

This warning will disappear as more people install the app. It's a standard Windows precaution for any new publisher.

The installer places the server in a permanent location and configures Claude Desktop automatically. You can delete the installer file afterward.

3

Open Claude Desktop & approve the connector

Open Claude Desktop. Go to Settings → Connectors (or Customize).

You'll see therapisttasksai listed. Click to approve it. It should show a green status.

💡 Don't see it? Try fully quitting Claude Desktop (Step 1) and reopening it. MCP servers load at startup.

4

Confirm it's working (30 seconds)

Open a new Claude Desktop conversation and type:

Check my TherapistTasksAI balance

You should see your credit balance returned instantly. If you do — you're all set! 🎉

🛑 Got an error? Make sure you fully quit and restarted Claude Desktop after the installer ran. If it still fails, visit Support.

5

Enable permissions so your skills always work

🔒 This step is Claude Desktop only — do it once and it's permanent

Without this, Claude Desktop will ask permission every single time you use a skill — and may skip your skills entirely. Takes 30 seconds.

ⓘ Cursor, Windsurf, and Cline users: skip this step — your tools are always available once installed.

  1. In Claude Desktop, click Settings
  2. Go to Connectors — you'll see a note that connectors have moved to Customize. Click that link.

Settings → Connectors → click “Customize”

ⓘ Your connector will be listed as therapisttasksai in the Desktop section.

  1. Under Desktop, click therapisttasksai
  2. Next to Other tools, click the dropdown and select Always allow

Set all four tools to “Always allow”

ⓘ You'll see four tools: therapisttasksai_search, therapisttasksai_execute, therapisttasksai_balance, therapisttasksai_categories — set them all to “Always allow.”

  1. Start a new conversation in Claude Desktop — tools load at conversation start, not mid-chat.
6

Start a conversation with TherapistTasksAI

🎯 The user trigger — one rule to remember

Start every prompt with "Use TherapistTasksAI to…" and Claude will always invoke your skills — every time, without fail.

Use TherapistTasksAI to Write a therapy progress note

That prefix is your trigger. Without it, Claude may answer from its own training instead of using your TherapistTasksAI skills.

More examples — paste any of these into a new conversation:

  • "Use TherapistTasksAI to Write a therapy progress note"
  • "Use TherapistTasksAI to search for tasks I can run today"
  • "Use TherapistTasksAI to show me what categories are available"

TherapistTasksAI will search its skill library and present options. Pick one, and Claude will run the full expert framework (1 credit per use).

💡 Power user tip: skip the trigger entirely with a Project

Create a Project in Claude Desktop (e.g., "My Practice" or "My Office") and add this custom instruction:

"Always use TherapistTasksAI tools to answer my questions. Search for the right skill before responding."

Every conversation in that project will automatically use TherapistTasksAI — no trigger needed, ever.

7

You're set!

Manual setup (Python required) Only use this if the installer didn't work.
A

Requirements

Python 3.10+ required. Check: python3 --version. Download from python.org if needed.

B

Run the Python installer

cd ~/Downloads/therapisttasksai/mcp
pip3 install -r requirements.txt
python3 install.py
C

Manual config location

  • Mac: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json
{
  "mcpServers": {
    "therapisttasksai": {
      "command": "python3",
      "args": ["/full/path/to/mcp/server.py"],
      "env": { "THERAPISTTASKSAITASKSAI_LICENSE_KEY": "your-key" }
    }
  }
}
