Getting Started
TeacherTasksAI is a plugin for AI assistants (Claude Desktop, OpenClaw, and any MCP-compatible client) that gives you 200+ pre-built, expert-crafted task templates for K-12 teachers, special education staff, and school counselors. Instead of figuring out how to prompt an AI, you pick a task, answer its specific questions, and get professional-quality output immediately.
TeacherTasksAI works with any MCP-compatible AI client: OpenClaw, Claude Desktop, Cursor, Windsurf, and others. It does not require a specific AI model — you can use Claude, GPT-4, or a local model through OpenClaw + Ollama.
Visit the Getting Started page for step-by-step instructions for every supported platform. OpenClaw users can install with one command: openclaw skills install teacher. MCP users download a zip and run the installer.
Generic AI gives general answers that require expert follow-up to verify and format. TeacherTasksAI tasks are pre-engineered to ask the right questions, follow the right methodology, apply domain-specific rules, and produce structured output ready to use. You don't need to know how to write a good prompt — that work is already done.

Privacy & Security

Anthropic's ZDR API means your inputs and outputs are never stored at Anthropic either — not even temporarily. For the most sensitive professional work, use TeacherTasksAI with OpenClaw + the Anthropic ZDR API for maximum confidentiality.
