🚀 Getting Started

1

Purchase Credits

Buy a credit pack on the pricing page. You'll receive your license key by email immediately after checkout.

2

Install Your AI Platform

TeacherTasksAI works with OpenClaw and Claude Desktop. Install your preferred platform if you haven't already.

OpenClaw Claude Desktop
3

Download the Skill Pack

After purchase, go to your confirmation page and download the TeacherTasksAI skill pack. Your license key activates the download.

4

Install & Run Your First Task

Place the skill pack in your OpenClaw skills folder (or follow the Claude Desktop install instructions). Then ask your AI to run a task. It's that simple.

💻 Supported Platforms

TeacherTasksAI is designed for use with AI platforms that support the OpenClaw skill format:

OpenClaw

The recommended platform. OpenClaw runs on Mac and Windows and provides the full TeacherTasksAI experience — task execution, credit management, and automatic updates. Visit openclaw.ai to install.

Claude Desktop

Anthropic's desktop app with MCP (Model Context Protocol) support. Install the skill pack as an MCP server. Visit claude.ai/download to get Claude Desktop.

My credits aren't showing up

Credits appear after payment confirmation, usually within seconds. If they don't appear after a few minutes, email us with your license key and we'll sort it out quickly.

Make sure you're running the latest version of the skill pack. If the issue persists, send us the task name and a brief description of what happened — we'll investigate.
