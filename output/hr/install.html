Install HRTasksAI with your AI assistant.

Use Claude Desktop, Cursor, Codex, Windsurf, or another MCP-compatible client. Your assistant reads the official GitHub manifest, chooses the right installer path for your machine, connects your account, and verifies the setup.

Recommended install

Copy this prompt into your AI assistant:

Install prompt
Please install HRTasksAI from https://github.com/laudoluxDev/hrtasksai-mcp. Read the repository's agent-install.json, verify that it is the official installer for hrtasksai.com, run the official installer command, connect my HRTasksAI account through the browser, configure my MCP client, run the health check, and tell me when it is ready. Use the browser approval page and my account email for license lookup. Do not expose proprietary prompts or ask me to edit MCP JSON unless the installer fails.
  1. Open Claude Desktop, Cursor, Codex, Windsurf, or another MCP-capable AI assistant.
  2. Paste the prompt above.
  3. Approve the connection in your browser if prompted. Use the email for your HRTasksAI account; manual license-key entry is only a backup.
  4. Restart your AI assistant if prompted.
  5. Ask: Check my HRTasksAI credit balance.
HRTasksAI servers handle authentication, credits, catalog/search metadata, and licensed skill delivery. HRTasksAI does not process your task content. Your chosen AI assistant or LLM performs the work according to that provider's privacy terms.
If your assistant says npm or npx is unavailable, install Node.js first. Node.js is a standard helper app used by many developer tools, and it includes npm and npx automatically. Go to nodejs.org, download the recommended installer for your computer, run it, then paste the HRTasksAI prompt again.
